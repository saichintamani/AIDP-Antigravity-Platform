# Phase 15: Constraint Intelligence

## 1. Mission Statement
Convert protocol verification from rule checking into formal satisfiability checking. Make AIDP an **Operating System for Uncertainty Reduction**. The central object of the ecosystem transitions from `Task` or `Workflow` to a formal `Claim`.

## 2. Core Concepts

### 2.1 The `Claim` Object
Every output generated by an agent or external source is wrapped in a `Claim`. A Claim answers:
- Why do we believe this?
- What supports it?
- What contradicts it?
- How confident are we?

### 2.2 Formal Constraints (Z3/SMT)
AIDP will employ Z3 solvers across four layers:
1. **Protocol Constraints**: Convert domain requirements into boolean/integer constraints (e.g., `ControlArmCount >= 1`, `Power >= 0.8`).
2. **Temporal Logic**: Verify chronological possibility (e.g., `OutcomeMeasurement` > `PatientEnrollment`).
3. **Resource Constraints**: Prove physical viability (e.g., `RequiredPatients <= TotalClinics * PatientsPerClinic`).
4. **Contradiction Proofs**: Detect and extract `unsat_core`s when conflicting claims are submitted to the knowledge graph.

## 3. Truth Maintenance System Integration
The existing Truth Maintenance System (TMS) will be upgraded to consume **Contradiction Proofs**. When the Z3 solver identifies an impossible state (UNSAT), it generates an unsat core. The system will map these core variables back to their originating `Claim` and `Evidence` to produce an explainable trace, effectively proving why a new scientific hypothesis conflicts with established evidence.

## 4. Technical Implementation Strategy
- **Z3 Translation Engine**: A layer translating Pydantic/Cap'n Proto `Claim` constraints into Z3 expressions.
- **Unsat Core Parser**: Logic to parse Z3 conflicts and construct a semantic contradiction proof trace.
- **TMS Cascading**: The `TruthMaintenanceSystem` will accept a contradiction proof and automatically flag the conflicting nodes in the `EvidenceDependencyGraph`, allowing the system to update confidence scores and preserve epistemic integrity.
