# THE ANTIGRAVITY PRIORITY STACK

If development is frozen for the next 12 months, the following improvements will create the highest increase in scientific credibility per hour of work:

| Priority | Area | Impact | Description |
| -------- | ---- | ------ | ----------- |
| **1** | **Real human evidence** | Extremely High | Deploy AlignEval surveys and collect actual human judgments. |
| **2** | **Real model evidence** | Extremely High | Wire the orchestrator to actually save full multi-agent transcripts, not just the final output. |
| **3** | **ConstraintBench validation** | Very High | Implement programmatic verification (Z3) against the LLM outputs to objectively score passes/fails. |
| **4** | **Historical replay publication** | Very High | Publish the results of the Prions and H. pylori replays using strict temporal cutoffs to prove the system works. |
| **5** | **Gemma + Ollama Full Compatibility** | High | Guarantee the entire system runs locally on open-weights (Gemma 2) without external APIs. This proves it is a reproducible research tool. |
| **6** | **Open-source readiness** | High | Add LICENSE, `requirements.txt`, setup docs, and clean up dead code so external researchers can fork and run it. |
| **7** | **Gemini API support** | Medium | Implement proper Gemini tool calling and structured outputs for high-context benchmarking. |
| **8** | **Agent optimization** | Medium | Collapse the 25 aesthetic agents into 3 highly rigorous functional agents (Generator, Devil's Advocate, Verifier). |
| **9** | **Additional UI work** | Low | The UI is already 10/10. More UI work yields diminishing returns for scientific credibility. |
| **10** | **More dashboards** | Very Low | A distraction from evidence collection. |
| **11** | **More agent layers** | Very Low | Pure token waste without increasing empirical validity. |

## Final Conclusion
The strongest audit outcome is not "Everything is perfect." It is: 
**"Here are the exact architectural weaknesses, missing evidence, readiness gaps, and next actions required to transform Antigravity from an impressive UI prototype into a trusted, reproducible scientific research platform."**
