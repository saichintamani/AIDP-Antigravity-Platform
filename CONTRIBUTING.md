# Contributing to AIDP

Welcome! AIDP is currently in an **Architecture Feature Freeze** following the Version 1.0 Scientific Validation Release. 

We are currently only accepting PRs that:
1. Fix bugs in the reasoning engine.
2. Add historical datasets to `DiscoveryBench`.
3. Improve evaluation metrics in the `ScientificEvaluationSuite`.

We are not accepting PRs for new cognitive architectures, new agent types, or major infrastructure changes until Phase F begins.
