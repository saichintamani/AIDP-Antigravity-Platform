# Phase 16: Scientific Introspection Engine

## 1. Mission Statement
Transform AIDP from a system that generates *Cognitive Objects* to a system that generates *Knowledge About Cognitive Objects*. By aggregating telemetry from Claims, Explanations, and Truth Maintenance, the Introspection Engine will discover meta-patterns in scientific reasoning, measure reviewer precision, and generate reusable lessons learned.

## 2. Core Deliverables

### 2.1 Failure Genome
Classifies every failure to create a living map of scientific uncertainty.
- Tracks rejection categories based on Z3 `unsat_core` proofs.
- Identifies the most common failure modes across domains.

### 2.2 Reviewer Performance Analytics
Tracks the precision of reviewer personas (e.g., Statistician, Ethicist, Engineer).
- Computes correlation between a reviewer's approval and the ultimate SAT/UNSAT outcome of a claim.
- Transitions reviewer trust from role-based to evidence-based.

### 2.3 Assumption Observatory
Monitors assumptions embedded within generated claims.
- Tracks historical outcomes: Supported, Contradicted, Unresolved.
- Quantifies the risk associated with common assumptions (e.g., "Protein A activates Protein B").

### 2.4 Constraint Observatory
Analyzes the performance of Z3 formal verification constraints.
- Identifies which constraints fail most often (e.g., temporal vs. statistical power).
- Provides a roadmap for improving upstream planners based on frequent constraint violations.

### 2.5 Learning Signals
The flagship output of the Introspection Engine.
- Generates `LessonLearned` objects based on statistically significant patterns.
- Example: *"Clinical protocols generated by Planner X frequently fail power analysis (Confidence: 0.91, Evidence: 247 observations)."*

## 3. Architecture
- **Observatory Models**: `FailureMode`, `ReviewerPrecision`, `AssumptionTracker`, `LessonLearned`.
- **Introspection Engine**: An aggregator that ingests `Claim` and `CausalExplanation` histories to compute the observatory metrics and emit learning signals.
